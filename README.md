A high-performance e-commerce application built with Flutter, cloning the core functionality of the Texnomart marketplace. This project demonstrates a production-ready architecture, efficient state management, and seamless integration with complex native features like Yandex Maps.

Tech Stack & Architecture
This project follows Clean Architecture principles to ensure code maintainability and scalability.

State Management: flutter_bloc (BLoC) for predictable state transitions.

Data Modeling: Freezed & Json Serializable for type-safe, immutable models.

Networking: Dio & Retrofit for type-safe REST API consumption.

Dependency Injection: GetIt for efficient service location.

Local Database: Hive (CE) for high-speed local caching and persistence.

Navigation: GoRouter for declarative, URL-based routing.

Maps & Location: Yandex Mapkit & Geolocator for store location and user tracking.

Key Features
Dynamic Product Catalog: Categorized listing with complex filtering using Retrofit.

Local Persistence: Cart and Wishlist functionality powered by Hive for offline access.

Advanced UI/UX:

Smooth carousels with carousel_slider.

Skeleton loaders using shimmer for better perceived performance.

Lottie animations for interactive user feedback.

Customized font family (Roboto) and SVG support.

Map Integration: Full Yandex Maps integration to find physical Texnomart branches.

Responsive Images: Optimized image loading with cached_network_image.
